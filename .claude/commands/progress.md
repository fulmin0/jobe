Generate a **comprehensive progress report**.

1. Read `progress/summary.md` for aggregate data.
2. Read `progress/question-log.md` for detailed question history.
3. Read recent session files from `progress/sessions/`.
4. Present a report with:
   - Total sessions completed and overall average
   - Average score by question type (table)
   - Score trends (improving/declining/stable)
   - Top strengths (dimensions consistently scoring 4+)
   - Weak areas in priority order (dimensions below 3)
   - Specific patterns identified
   - Top 3 recommended focus areas for next sessions
   - Last 5 sessions summary

**Period:** $ARGUMENTS (default to "all". Options: week, month, all)
